# ADR-006: ASP型ワイズマンの自動化戦略

## ステータス
**Superseded (2026-03-23)**

> **廃止理由**: クライアント実環境のスクリーンショットにより、使用中のワイズマンシステムSPが
> **オンプレミス版（ネイティブWindowsデスクトップアプリ）**であることが確定した。
> ASP版（ブラウザベース）の自動化戦略は本クライアントには適用されない。
>
> ネイティブアプリの自動化戦略はADR-001（Revised）に記載。

> **参考**: ワイズマンにはASP版も存在する。将来ASP版クライアントを扱う場合は
> 本ADRの内容（Playwright + ブラウザ自動化）が参考になる。
