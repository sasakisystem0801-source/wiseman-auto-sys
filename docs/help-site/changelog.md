## 凡例

| 種別 | 意味 |
|---|---|
| 🆕 追加 | 新機能 / 新ページ |
| ✏️ 修正 | 既存内容の更新・改善 |
| 🐛 不具合修正 | 誤記・誤情報の訂正 |

## バージョニング方針

[Semantic Versioning](https://semver.org/lang/ja/) に準拠。

- **Major (X.0.0)**: 全面改訂、大幅な構成変更
- **Minor (1.X.0)**: 機能ページ追加 / 大幅な内容追記
- **Patch (1.0.X)**: 誤記修正 / 軽微な改善
