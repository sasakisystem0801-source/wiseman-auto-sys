# 実機確認チェックリスト

このマニュアルは macOS 環境で **実装コードから推測** して作成された箇所を含みます。  
TeamViewer で本田様 PC に接続できた際、以下を確認して差分があれば修正してください。

確認後は `changelog.md` に修正内容を追記して、Patch version を bump（例: 1.0.0 → 1.0.1）してください。

---

## 確認方法

```bash
# ローカルプレビュー（macOS）
cd docs/help-site && python3 -m http.server 4567
# → http://localhost:4567/ で編集中の状態を確認

# 修正後のデプロイ
cd /Users/yyyhhh/Projects/wiseman_auto_sys
firebase deploy --only hosting:help --project wiseman-hub-prod
```

## 修正のワークフロー

1. 該当 Markdown を編集（誤記訂正 / 文言差し替え / スクショ追加）
2. ローカルプレビューで確認
3. `_meta.json` の `manual_version` を Patch bump（1.0.0 → 1.0.1）+ `manual_updated` を本日日付に
4. `changelog.md` の先頭に新エントリを追記
5. `firebase deploy --only hosting:help --project wiseman-hub-prod`
6. ブラウザで反映確認
