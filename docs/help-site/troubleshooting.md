# トラブルシューティング

## 症状別の対処方法

### アプリが起動しない

| 確認項目 | 対処 |
|---------|------|
| ショートカットアイコンが正しいか | デスクトップの「Wiseman PDF ツール」アイコンか |
| エラーダイアログが出るか | エラーメッセージを撮影して開発担当へ連絡 |
| PC を再起動したか | 再起動後に再度試す |
| アンチウイルスがブロックしていないか | アンチウイルスの履歴を確認 |

### 起動はするが動作が遅い

| 原因の可能性 | 対処 |
|------------|------|
| 大量データ処理中 | 30 秒～1 分待つ |
| ネットワーク不調（GCP 同期失敗） | ネット接続を確認 |
| 他のアプリが CPU 使用中 | タスクマネージャーで確認 |
| PC のメモリ不足 | 他のアプリを閉じる |

### スプレッドシートが読めない

| 原因 | 対処 |
|-----|------|
| ネットワーク接続不可 | インターネット接続確認 |
| スプレッドシート ID が間違い | 設定画面で再確認 |
| スプレッドシートへのアクセス権がない | スプレッドシート所有者に共有依頼 |
| シート名が変わった | 「シート一覧更新」を実行 |

### ⚠ マークが大量に出る

| 機能 | ⚠ の意味 | 対処 |
|-----|---------|------|
| B 配置 | 居宅マッピング未登録 | スプレッドシートに居宅追記 |
| B 配置 | 利用者フォルダ未発見 | FAX 事業所フォルダ内に利用者フォルダ作成 |
| B 配置 | 月別 PDF 不在 | カルテで該当月 PDF 出力確認 |
| C 配置 | xlsx パス未設定 | 該当行を右クリック → xlsx ピッカーで設定 |
| C 配置 | 利用者シート不在 | xlsx 内のシート名確認 |

### Excel が起動しない（C 配置）

| 原因 | 対処 |
|-----|------|
| Excel がインストールされていない | 開発担当へ連絡（Excel 必須） |
| Excel が他のファイルでフリーズ中 | Excel を完全終了 → 再実行 |
| Excel の自動起動が拒否されている | アンチウイルス設定確認 |

### PDF 配置先がおかしい

| 確認項目 | 対処 |
|---------|------|
| 設定で FAX 事業所ルートが正しいか | 設定画面確認 |
| 事業所判定の alias が古い | alias 設定更新 |
| 配置先フォルダの存在確認 | エクスプローラーで確認 |

### Tera-station ネットワークドライブにアクセスできない

| 原因 | 対処 |
|-----|------|
| ネットワーク接続不可 | LAN ケーブル / Wi-Fi 確認 |
| ネットワークドライブ未マウント | エクスプローラーで `\\Tera-station\share` を確認 |
| 認証情報の期限切れ | PC 管理者に連絡 |

## エラーメッセージ別の対処

### `設定不足: 設定でスプレッドシートIDを登録してください`

→ [⑤ 設定](guide/settings.md) でスプレッドシート ID を入力してください。

### `設定編集不可`

→ 実行中は設定変更できません。実行完了を待ってから再度試してください。

### `設定再読込失敗`

→ ネットワーク接続を確認。改善しない場合は開発担当へ連絡。

### `ファイルが他のプロセスで使用中`

→ xlsx ファイルを Excel で開いていないか確認。閉じてから再実行。

---

## 困ったらやること

1. **エラーメッセージのスクリーンショット** を撮る
2. **どの機能** を **どの順番で** 操作したか覚えておく
3. 開発担当（佐々木）へ連絡:
   - スクリーンショット
   - 操作手順
   - 発生時刻
   - その他気づいたこと

---

## 緊急時の対応

### 業務時間中にアプリが完全に動かない

1. アプリを **強制終了**（タスクマネージャー → wiseman_hub.exe → 終了）
2. PC を再起動
3. 再度アプリを起動
4. それでもダメな場合は **手作業** で業務を継続しつつ、開発担当へ連絡

### 誤って大量のファイルを配置してしまった

1. **作業を中止**（実行中なら閉じる）
2. **何が起こったか** をメモ
3. 開発担当へ連絡（即時対応します）

> **重要**: Tera-station の `\\Tera-station\share\trashbox\` に削除ファイルが残ることがあります。  
> 開発担当に確認すれば復旧可能な場合があります。
