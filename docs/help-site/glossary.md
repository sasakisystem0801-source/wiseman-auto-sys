# 用語集

業務 / システム両面でよく出てくる用語の解説です。

## 業務用語

### B 帳票（運動機能向上計画書）

利用者ごとの運動機能向上計画と評価を記録する帳票。  
月次のモニタリング対象者に対して作成・配布されます。

### C 帳票（経過報告書）

利用者の経過を担当者が記載する報告書。  
担当者ごとに xlsx ファイルで管理され、月次で利用者シートを抽出して配布します。

### モニタリング

利用者の状態を定期的に評価する業務。  
B 帳票のモニタリング日付がある利用者が当月の配置対象となります。

### 居宅（居宅介護支援事業所）

利用者の介護プランを作成する事業所。  
B 配置時に「利用者 → 居宅」のマッピングが必要です。

### FAX 事業所

報告書を FAX で送信する送信先事業所。  
通常 `\\Tera-station\share\03.FAX(事業所)` 配下に事業所別フォルダが配置されています。

### カルテ

利用者の月別 PDF が保管されている場所。  
B 配置時にカルテから月別 PDF を取得します。

## システム用語

### .ex_ ファイル

Wiseman 独自の拡張子。中身は PDF データだが、そのままでは開けない。  
このツールで `.pdf` に変換します。

### Wiseman

介護記録ソフト「ケア記録 Wiseman システム SP」。  
USB ドングル認証の Windows ネイティブアプリ。

### スプレッドシート

Google スプレッドシートのこと。  
月別のシート（タブ）で利用者情報を管理しています。

### 担当者マッピング

担当者名 → xlsx ファイルパスの対応表。  
C 配置時に使用されます。

### alias 設定

事業所名の表記揺れマッピング（例: 「A 事業所」「Ａ事業所」を同一視）。  
ex_ 振り分け時の事業所判定で使用されます。

### GCP / Google Cloud

Google Cloud Platform。  
本ツールの設定同期、リリース配信に使用しています。

### GCS（Google Cloud Storage）

GCP のクラウドストレージ。  
担当者マッピング等のキャッシュを同期しています。

### USB ドングル

Wiseman 本体の認証に使う物理 USB デバイス。  
本ツールでは使用しませんが、Wiseman 起動時に必要です。

### Tera-station

NAS（ネットワーク接続ストレージ）の製品名。  
FAX 事業所フォルダや trashbox（削除復旧領域）が配置されています。

## 操作画面の用語

### ダイアログ

メイン画面の上に表示される **小さな別ウィンドウ**。  
各機能ボタンをクリックすると開きます。

### シート一覧更新

スプレッドシートから **タブの一覧** を取得し直す操作。

### 対象行を読込

選択中のシートから **配置対象の利用者行** を抽出する操作。

### 配置

PDF を FAX 事業所フォルダ配下の利用者フォルダにコピーする操作。

### 結合

複数の PDF を 1 つの PDF にまとめる操作。
