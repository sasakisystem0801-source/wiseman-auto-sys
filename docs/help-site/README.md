# Wiseman PDF ツール 操作マニュアル

このマニュアルは、Wiseman PDF ツール（デスクトップアプリ）の操作方法を説明します。

## このツールでできること

業務フロー順に 5 つの機能があります。

| # | 機能 | 役割 |
|---|------|------|
| ① | **ex_ファイル変換 + 振り分け** | Wiseman から出力した `.ex_` ファイルを PDF に変換し、事業所フォルダに自動振り分け |
| ② | **B: 運動機能向上計画書 自動配置** | スプレッドシートの月別データを元に、運動機能向上計画書 PDF を利用者フォルダに自動配置 |
| ③ | **C: 経過報告書 自動配置** | スプレッドシートの担当者情報を元に、経過報告書 xlsx を PDF 化して事業所フォルダに自動配置 |
| ④ | **事業所フォルダ一括結合** | 事業所フォルダ内の PDF を一括結合（複数事業所をまとめて処理可） |
| ⑤ | **設定** | スプレッドシート ID、対象月、フォルダパス等の設定変更 |

## 起動方法

デスクトップの **「Wiseman PDF ツール」** アイコンをダブルクリックします。

起動すると、5 つのボタンが業務フロー順に並んだメイン画面が表示されます。
